# nextjs-web — Skill de Claude Code

Skill instalable para **Claude Code**: dominio de Next.js 16 con App Router
(páginas, rutas, RSC, server actions, caching, i18n, testing y performance).
Autocontenida: funciona en cualquier proyecto Next.js.

## Cómo instalar

1. Descomprime este ZIP. Obtendrás una carpeta `nextjs-web/`.
2. Cópiala a tu directorio de skills de Claude Code:
   - **Personal (todos tus proyectos):** `~/.claude/skills/nextjs-web/`
   - **Solo un proyecto:** `<tu-proyecto>/.claude/skills/nextjs-web/`
3. Reinicia Claude Code. La skill se activará sola cuando trabajes con Next.js,
   o invócala por su nombre.

## Contenido

- `SKILL.md` — guía principal (stack, arquitectura, workflow, antipatrones).
- `references/` — app-router, components, i18n, performance, testing.
