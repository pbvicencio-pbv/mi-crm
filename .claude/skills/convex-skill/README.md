# convex-skill — Skill de Claude Code

Skill instalable para **Claude Code**: dominio de Convex, el backend reactivo en
TypeScript (base de datos + funciones de servidor + sync en tiempo real). Queries,
mutations, actions, schemas, auth (Clerk/Convex Auth), agentes IA/RAG, storage,
crons, HTTP endpoints y testing. Autocontenida.

## Cómo instalar

1. Descomprime este ZIP. Obtendrás una carpeta `convex-skill/`.
2. Cópiala a tu directorio de skills de Claude Code:
   - **Personal:** `~/.claude/skills/convex-skill/`
   - **Solo un proyecto:** `<tu-proyecto>/.claude/skills/convex-skill/`
3. Reinicia Claude Code. Se activará sola al trabajar con Convex.

## Contenido
- `SKILL.md` — guía principal.
- `references/` — functions, schema-and-data, auth, frontend, ai-and-search, cli-and-ops, advanced.
- `scripts/` — helper.
